# Spring Oscillation Model

A Python-based  simulation of a spring–mass oscillation system, developed to investigate vibration behaviour, damping, energy transfer, and resonance.

## Overview

This project models the motion of a mass attached to a spring system using numerical methods. 
The scenario was inspired by questiom 18 of the 2024 Paper 1 AQA A-Level Further Maths paper.
The model solves the governing differential equations of motion and visualises how system parameters affect the resulting oscillations.

The simulation includes:
- Spring restoring forces
- External inputs
- Damping effects
- Energy transfer within the system
- Resonance behaviour

The model was developed to explore concepts relevant to mechanical vibration analysis, such as natural frequency, forced oscillations, and energy dissipation.

## Model Description

The system is represented by a second-order ordinary differential equation:

\[
m\ddot{x} + c\dot{x} + kx = F(t)
\]

where:

- \(m\) = mass
- \(c\) = damping coefficient
- \(k\) = spring stiffness
- \(x\) = displacement
- \(F(t)\) = external forcing function

The differential equation is solved numerically using `scipy.integrate.solve_ivp`.

## Features

### Solution Approximation
The model calculates displacement and velocity over time for a given set of parameters.

### Energy Analysis
The simulation calculates:
- Kinetic energy
- Elastic potential energy
- Gravitational potential energy
- Energy lost through damping

This allows investigation of energy conservation and dissipation within the system.

### Animation
The model can produce an animation to better visualise the results
